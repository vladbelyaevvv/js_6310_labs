# Разработать компоненты:
1. Отображение следующего тетрамино
2. Отображение стакана с заполненынми фигурами
## Функциональные требования
1. Следующее тетрамино:
- принимает объект тетрамино (который содержит его координаты в сетке, цвет по умолчанию и другие необходимые параметры)
- отображает тетрамино
2. Стакан с фигурами
- Имеет заданную дефолтную размерность тетриса
- Принимает двумерный массив чисел
- Каждое число - отвечает за цвет ячейки в стакане
## Нефункциональные требования
- Запрещено использовать сторонние UI библиотеки

# Разработать компонент отображения топологии сети
## Функциональные требования
- компонент должен отображать топологию сети (компьютеры, узлы, адреса)
- компонент должен корректно масштабироваться при изменении ширины страницы
- компонент должен иметь возможность зума топологии
## Нефункциональные требования
- Допустимо использование библиотеки для отображения графа React Flow

# Разработать компонент кейса с результатами работ (портфолио)
## Функциональные требования
- Карточка должна содержать информацию об названии работы
- Карточка должна содержать описание и ключевые характеристики
- Карточка картинку
- При клике на карточку должна отображаться возможность отобразить новость целиком.Допускается отобразить картинки Галереей внизу или наверху новости
- Карточка должна содержать две кнопки: для просмотра кейса подробнее, для заказа
- Все вышеперечисленные параметры карточки передаются через props компонента
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
